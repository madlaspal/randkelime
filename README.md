# 🇪🇸 RandKelime

RandKelime is a simple web application that displays random Spanish words along with their Turkish meanings.

## ✨ Features

- 🎲 Random Spanish word generator
- 🇹🇷 Turkish translations
- 📚 JSON-based word database
- ⚡ Fast and lightweight
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- JSON

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/madlaspal/randkelime.git
```

2. Open the project

```text
index.html
```

or run it using **Live Server** in Visual Studio Code.
